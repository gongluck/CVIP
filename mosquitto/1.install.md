## 下载安装

```shell
apt-get update
apt-get install mosquitto
```

## 开启服务

```shell
systemctl start mosquitto
```

## 停止服务

```shell
systemctl stop mosquitto
```

## 重启服务

```shell
systemctl restart mosquitto
```

## 添加用户

```shell
mosquitto_passwd -c /etc/mosquitto/pwfile.txt [user]
```
